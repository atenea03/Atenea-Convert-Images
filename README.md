## 🚀 How to Use

**Step 1 — Open the program**
- Double-click `ate_convert_images.exe`.
- The Atenea window will open with a dark background and golden logo.

**Step 2 — Select input images**
- **📁 Select folder** → processes all matching images inside a folder.
- **🖼 Select files** → pick individual image files.
- Only files matching the chosen input format will be processed.

**Step 3 — Select output folder**
- Click **Select folder** under Output folder.
- Converted files will be saved there. Default folder name: `converted`.

**Step 4 — Choose formats**
- **From:** filter which format to read (`PNG`, `JPG`, `WEBP`, `BMP`, `TIFF`, or `All`).
- **To:** choose the format to convert to.

**Step 5 — Adjust quality** *(WEBP and JPG only)*
- A quality slider appears when WEBP or JPG is selected as output.
- Recommended value: **85** — good balance between quality and file size.

**Step 6 — Convert**
- Click the orange **Convert** button.
- A progress bar will show the conversion in real time.
- When finished, a popup will show a summary of converted files and errors.

## 🗂️ Supported Formats

| Format | Extensions | Notes |
|--------|------------|-------|
| PNG | `.png` | Lossless. Supports transparency. |
| JPG | `.jpg` / `.jpeg` | Lossy. Transparent areas become white. |
| WEBP | `.webp` | Lossy. Good balance of quality and size. |
| BMP | `.bmp` | Uncompressed. Transparent areas become white. Large files. |
| TIFF | `.tiff` / `.tif` | Lossless. High quality. Large file size. |

---

## 💡 Tips

- Converting to **JPG or BMP** replaces transparent areas with a white background.
- Converting to **PNG** preserves transparency.
- Select **All** in the From dropdown to process mixed formats at once.
- The output folder is created automatically if it does not exist.
- Filenames are preserved — only the extension changes.
